---

## 서비스 소개

**CertPass**는 자격증 시험 준비에 특화된 온라인 강의 플랫폼입니다.
수험생은 웹 · 모바일 환경에서 자격증 분야별 강의를 탐색하고, 수강 신청 후 과목별 학습 진도를 관리할 수 있습니다.

---

## 프로젝트 단계별 계획 (P1 ~ P3)

### P1 — MVP (현재 단계)

**목표:** 핵심 학습 기능 구현 (고객 입장에서의 기본 흐름 완성)

| 영역 | 구현 범위 |
|------|----------|
| 프론트엔드 (Web) | 강의 목록/상세, 수강 신청, 강의 재생, 진도 관리 |
| 프론트엔드 (Mobile) | 반응형 웹 (모바일 최적화 UI) |
| 백엔드 | 회원가입/로그인, 강의 CRUD, 수강 등록, 진도 저장 API |
| 데이터베이스 | 사용자, 강의, 강의 섹션/영상, 수강 내역, 진도율 |

### P2 — 역할 분리 및 전체 기능 완성

**목표:** 관리자·강사·수강생 역할 분리, 인증 강화, 전체 기능 구현

- JWT Refresh Token 및 OAuth 소셜 로그인 도입
- 강사 대시보드 (강의 등록·수정·통계)
- 관리자 페이지 (회원 관리, 강의 승인)
- 강의 검색 / 자격증 분야 필터링 / 리뷰·평점
- 모의고사 / 기출문제 풀이 기능

### P3 — 운영 및 최적화

**목표:** 결제·알림·모니터링 도입, 코드 리팩토링 및 성능 최적화

- PG 연동 결제 (강의 유료화)
- 시험 D-day 알림 / 이메일 / 푸시 알림
- 로깅, 모니터링 (Prometheus + Grafana 등)
- 스케줄링 / 배치 작업
- 기존 코드 리팩토링 및 성능 최적화

## P1 기획 상세

### 1. 타깃 사용자 (고객 관점)

| 사용자 유형 | 설명 |
|------------|------|
| 수험생 (일반 사용자) | 자격증 취득을 목표로 강의를 탐색하고 수강 신청 후 영상을 시청하며 학습하는 주 사용자 |

> P1에서는 수험생 단일 역할만 지원. P2에서 강사/관리자 역할 추가 예정.

---

### 2. 주요 기능 (P1 범위)

#### 회원 기능
- 회원가입 / 로그인 / 로그아웃
- 내 정보 조회 및 수정 (프로필, 비밀번호)

#### 강의 탐색
- 강의 목록 조회 (자격증 분야별 필터)
- 강의 상세 정보 조회 (과목 커리큘럼, 강사 소개)

#### 수강 기능
- 강의 수강 신청 (무료 강의)
- 수강 강의 목록 조회 (내 강의실)

#### 학습 기능
- 강의 영상 재생 (과목/에피소드 단위)
- 학습 진도 저장 및 조회 (에피소드별 시청 완료 여부)
- 전체 강의 진도율 표시

### 3. 기술 스택

| 구분 | 기술 |
|------|------|
| 프론트엔드 | React 18, TypeScript, Vite, Tailwind CSS v4.1, React Query |
| 백엔드 | NestJS, TypeScript |
| 데이터베이스 | MongoDB (Mongoose) |

| 영상 스트리밍 | HTML5 Video (YouTube iframe 임베드) |

---

### 4. 화면 구성 (P1)

| 페이지 | 설명 |
|--------|------|
| 홈 | 인기 강의 목록, 자격증 분야 탐색 |
| 강의 목록 | 전체/분야별 강의 카드 리스트 |
| 강의 상세 | 강의 소개, 과목별 커리큘럼, 수강 신청 버튼 |
| 내 강의실 | 수강 중인 강의 목록 및 진도율 |
| 강의 플레이어 | 영상 재생 + 과목 목차 사이드바 |
| 회원가입 / 로그인 | 폼 기반 인증 |
| 마이페이지 | 프로필 조회·수정 |

---

### 5. 비기능 요구사항 (P1)

- 모바일 반응형 지원 (모바일 우선 설계)
- API 응답 시간 목표: 평균 300ms 이하
- 비밀번호 bcrypt 해싱 저장
- 환경 변수(.env)로 민감 정보 분리 관리
- 영상은 YouTube iframe 임베드 방식으로 제공 (별도 스토리지 불필요)
