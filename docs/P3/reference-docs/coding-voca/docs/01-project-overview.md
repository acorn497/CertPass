# Coding Voca — 프로젝트 개요

## 프로젝트 비전

**Coding Voca**는 코딩 초보자와 어린 학생 개발자들을 위한 **코딩 영어 단어장** 웹 애플리케이션입니다.

프로그래밍에서 사용하는 영어 단어(`return`, `class`, `object`, `string` 등)의 **한국어 발음**, **일상 대화에서의 뜻**, **코딩 세계에서의 뜻** 차이를 명확하고 재미있게 설명합니다.

---

## 핵심 가치

| 가치 | 설명 |
|---|---|
| 🎯 **명확함** | 일상 뜻 vs 코딩 뜻을 한눈에 비교 |
| 🗣️ **발음 가이드** | 정확한 한국어 발음 표기 (예: `variable` → "배리어블") |
| 🎮 **재미** | 어린 학생도 흥미를 느낄 수 있는 친근한 디자인 |
| 📱 **접근성** | 모바일/태블릿/PC 모두 지원 |
| 🔧 **확장성** | 단어 추가, 다국어, 관리자 기능 |

---

## 대상 사용자

### 학습자 (Public)
- **코딩 초보 학생** (초등 고학년 ~ 중학생)
- **성인 코딩 입문자**: 비전공자, 부트캠프 수강생
- **코딩 교육자**: 수업 보조 자료로 활용

### 관리자 (Admin)
- 단어 추가/수정/삭제
- 카테고리 관리
- 지원 언어 관리

---

## 핵심 기능

### 학습자 기능
| ID | 기능 | 설명 |
|---|---|---|
| F1 | 단어 카드 뷰 | 영어 단어 + 발음 + 일상/코딩 뜻 비교 카드 |
| F2 | 카테고리 탐색 | 기초문법, 데이터타입, OOP, 웹, 기타 등 |
| F3 | 검색 | 영어/한국어 발음/뜻으로 검색 |
| F4 | 단어 상세 | 발음, 일상 뜻(예문), 코딩 뜻(코드 예시), 관련 단어 |
| F5 | 언어별 필터 | 프로그래밍 언어별 관련 단어 필터링 |
| F6 | 퀴즈 | "이 단어의 코딩 뜻은?" 퀴즈 (향후 확장) |

### 관리자 기능
| ID | 기능 | 설명 |
|---|---|---|
| A1 | 단어 CRUD | 단어 추가/수정/삭제 |
| A2 | 카테고리 관리 | 카테고리 추가/수정/삭제 |
| A3 | 프로그래밍 언어 관리 | 지원 언어 추가/관리 |
| A4 | 대시보드 | 전체 통계 (단어 수, 카테고리 수 등) |
| A5 | 관리자 인증 | 로그인/로그아웃 |
