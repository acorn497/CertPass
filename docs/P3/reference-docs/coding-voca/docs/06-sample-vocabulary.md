# Coding Voca — 샘플 단어 목록

> 초기 시드 데이터로 사용할 56개 단어입니다.
> 카테고리별 영어 단어, 한국어 발음, 일상 뜻, 코딩 뜻을 정리합니다.

---

## ⌨️ 기초 문법 (basic-syntax) — 12개

| 단어 | 발음 | 일상 뜻 | 코딩 뜻 |
|---|---|---|---|
| variable | 배리어블 | 변하기 쉬운 | 변수 — 값을 저장하는 공간 |
| function | 펑션 | 기능, 역할 | 함수 — 특정 작업을 수행하는 코드 묶음 |
| return | 리턴 | 돌아가다 | 반환 — 함수가 결과 값을 돌려주는 것 |
| loop | 루프 | 고리, 반복 | 반복문 — 코드를 여러 번 실행 |
| condition | 컨디션 | 조건, 상태 | 조건문 — 참/거짓에 따라 다른 코드 실행 |
| constant | 컨스턴트 | 변함없는 | 상수 — 바꿀 수 없는 값 |
| assign | 어사인 | 배정하다 | 대입 — 변수에 값을 넣는 것 |
| declare | 디클레어 | 선언하다 | 선언 — 변수/함수를 만들겠다고 알리기 |
| execute | 엑서큐트 | 실행하다 | 실행 — 코드를 작동시키는 것 |
| statement | 스테이트먼트 | 진술, 성명 | 문(文) — 하나의 명령 |
| expression | 익스프레션 | 표현, 표정 | 식(式) — 값을 만드는 코드 조각 |
| parameter | 파라미터 | 매개변수 | 매개변수 — 함수 입력 값의 이름 |

## 📊 데이터 타입 (data-types) — 10개

| 단어 | 발음 | 일상 뜻 | 코딩 뜻 |
|---|---|---|---|
| string | 스트링 | 끈, 줄 | 문자열 — 텍스트 데이터 |
| integer | 인티저 | 정수(수학) | 정수형 — 소수점 없는 숫자 |
| float | 플로트 | 뜨다 | 실수형 — 소수점이 있는 숫자 |
| boolean | 불리언 | (논리학자 이름) | 불리언 — true/false |
| array | 어레이 | 배열, 진열 | 배열 — 여러 값을 순서대로 담는 구조 |
| null | 널 | 무효의 | null — "값이 없음" |
| undefined | 언디파인드 | 정의되지 않은 | undefined — 값이 아직 미정 |
| character | 캐릭터 | 인물, 성격 | 문자(char) — 하나의 글자 |
| type | 타입 | 유형, 종류 | 자료형 — 데이터의 종류 정의 |
| list | 리스트 | 목록, 명단 | 리스트 — 순서 있는 데이터 모음 |

## 🏗️ 객체지향 (oop) — 10개

| 단어 | 발음 | 일상 뜻 | 코딩 뜻 |
|---|---|---|---|
| object | 오브젝트 | 물체, 대상 | 객체 — 속성+기능을 묶은 단위 |
| class | 클래스 | 학급, 등급 | 클래스 — 객체의 설계도 |
| method | 메서드 | 방법, 수단 | 메서드 — 객체의 동작(함수) |
| property | 프로퍼티 | 재산 | 속성 — 객체가 가진 데이터 |
| inherit | 인헤릿 | 상속하다 (유산) | 상속 — 부모 기능을 물려받기 |
| instance | 인스턴스 | 사례, 경우 | 인스턴스 — 클래스로 만든 실제 객체 |
| constructor | 컨스트럭터 | 건설자 | 생성자 — 객체 생성 시 자동 실행 함수 |
| abstract | 앱스트랙트 | 추상적인 | 추상 — 구현 없이 틀만 정의 |
| interface | 인터페이스 | 접점, 경계면 | 인터페이스 — 클래스 규칙(약속) |
| polymorphism | 폴리모피즘 | 다형성(생물) | 다형성 — 같은 이름이 다르게 동작 |

## 🌐 웹 개발 (web-dev) — 9개

| 단어 | 발음 | 일상 뜻 | 코딩 뜻 |
|---|---|---|---|

| request | 리퀘스트 | 요청, 부탁 | 요청 — 서버에 데이터를 달라고 보내기 |

| cookie | 쿠키 | 과자 | 쿠키 — 브라우저에 저장되는 작은 데이터 |
| cache | 캐시 | 은닉처 | 캐시 — 빠른 접근 위한 임시 저장 |
| route | 라우트 | 경로, 길 | 라우트 — URL과 페이지 연결 규칙 |
