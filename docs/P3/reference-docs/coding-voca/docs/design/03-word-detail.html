variable
🗣️ 배리어블 /ˈvɛr.i.ə.bəl/
⌨️ 기초 문법 ⭐ 초급
🌍 일상에서는
🌦️
변하기 쉬운,
가변적인
예문
"The weather is very variable today."
오늘 날씨가 매우 변덕스럽다.
💻 코딩에서는
📦
변수 —
값을 저장하는 공간
설명
이름표가 붙은 상자처럼, 숫자·이름·참거짓 같은 데이터를 담아두는 공간이에요. 프로그램이 실행되는 동안 값이 바뀔 수 있어요.
💡 일상의 "변하는 것" = 코딩의 "변하는 값"
🐍 Python 
# 변수를 만들고 값을 저장하기
name = "홍길동"         # 이름을 담는 변수
age  = 15             # 나이를 담는 변수
is_student = True    # 학생인지 나타내는 변수

# 변수 값은 나중에 바꿀 수 있어요
age = 16              # 한 살 더 먹었어요!

print(name, "은", age, "살")
# 출력: 홍길동 은 16 살
name, age, is_student 모두 변수(variable)예요. 각각 문자열(string), 숫자(integer), 참거짓(boolean) 값을 저장하고 있어요. age = 16처럼 같은 변수에 새 값을 넣으면 이전 값은 사라져요.
