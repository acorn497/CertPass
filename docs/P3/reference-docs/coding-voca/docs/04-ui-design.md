## 학습자 페이지 구조

```
홈 (/)
├── 단어 목록 (/words) .............. 검색 + 필터 + 그리드
│   └── 카테고리/언어/난이도 필터
├── 단어 상세 (/words/:id) ......... 핵심 페이지
├── 카테고리 (/categories) ......... 카테고리 카드 그리드
│   └── 카테고리별 (/categories/:id)
├── 프로그래밍 언어 (/languages) ... 언어 카드 그리드
│   └── 언어별 (/languages/:id)
└── 소개 (/about)
```

## 일반 사용자 기능 구조 (Phase 9~11)

```
로그인 (/login) ..................... 사용자 로그인
회원가입 (/register) ................ 사용자 회원가입
│
내 단어장 목록 (/my/wordbooks) ...... 단어장 카드 그리드 + 새 단어장 생성
├── 단어장 상세 (/my/wordbooks/:id)
│   ├── 담긴 단어 목록 (카드 또는 테이블 뷰)
│   ├── 단어 제거 버튼
│   └── [퀴즈 시작] 버튼
│       ↓
│   퀴즈 설정 (/my/wordbooks/:id/quiz)
│   ├── 모드 선택 (의미 맞추기 / 단어 맞추기 / 혼합)
│   └── 문항 수 선택 (5 / 10 / 20 / 전체)
│       ↓
│   퀴즈 진행 (/my/wordbooks/:id/quiz/play)
│   ├── 프로그레스바 + 문항 번호
│   ├── 의미 맞추기: 4지선다 카드
│   └── 단어 맞추기: 텍스트 입력
│       ↓
│   결과 화면 (/quiz/:sessionId/result)
│   ├── 점수 원형 차트 / 퍼센트
│   ├── 정답/오답 목록
│   ├── [다시 하기] [단어장으로]
│   └── [오답 단어만 모아 다시 하기]
└── 퀴즈 기록 (/my/history) .......... 과거 퀴즈 결과 목록
```

### 단어 담기 버튼 (AddToWordbookButton)

**단어 목록 카드 (`/words`)**:
```
┌────────────────────────────────┐
│  [♡]  array  어레이             │   ← 우상단 북마크 아이콘 (로그인 시 활성)
│  배열 — 여러 값을 순서대로...    │
│  📁 기초문법    ⭐ 초급          │
└────────────────────────────────┘
```
클릭 시 → 내 단어장 목록 드롭다운 → 선택 → 토스트 "단어장에 추가됨" ✅

**단어 상세 (`/words/:id`)**:
```
│  💎 array                 [+ 단어장에 추가] │
│  어레이  /əˈreɪ/                           │
```

## 관리자 페이지 구조

```
관리자 로그인 (/admin/login)
│
관리자 대시보드 (/admin)
├── 단어 관리 (/admin/words) ........... 테이블 + CRUD
│   ├── 단어 추가 (/admin/words/new)
│   └── 단어 수정 (/admin/words/:id/edit)

├── 언어 관리 (/admin/languages) ....... 테이블 + CRUD

```

## 핵심 인터랙션

| 요소 | 효과 |
|---|---|
| 카드 호버 | `translateY(-4px)` + 글로우 쉐도우 |
| 검색 | 실시간 드롭다운 (debounce 300ms) |
| 뜻 비교 | 일상→코딩 전환 애니메이션 |
| 스크롤 | 페이드-인 나타남 (Intersection Observer) |
