UI Draft — 소모품 재고 관리 앱

모바일 기준 (390px) · 각 파일을 브라우저에서 열어 확인하세요

01

홈 화면

사용기한 경고 섹션 + 상품 목록 + 검색바 + FAB

 02

등록 방법 선택

Bottom Sheet — 바코드 스캔 / 직접 입력 선택

 03

바코드 스캔

카메라 뷰파인더 + 스캔 가이드 + 수동 입력 링크

 04

새 상품 등록 (바코드)

바코드 스캔 후 상품명 + 첫 재고 정보 입력

 05

새 상품 등록 (수동)

바코드 없는 상품 — 이름 직접 입력 + 무기한 토글 ON

 06

기존 상품 재고 추가

동일 바코드 스캔 시 — 기존 상품에 새 재고 추가

 07

상품 상세

상품 정보 + 재고 항목 목록 (수량 빠른 조절)

 08

재고 항목 수정

수량, 사용기한, 저장 위치, 메모 수정 + 삭제

 09

삭제 확인

상품 삭제 시 연관 재고 함께 삭제 경고 다이얼로그

 10

검색 결과

이름 텍스트 검색 — 매칭 키워드 하이라이트

 11

빈 상태

최초 진입 시 — 상품 등록 안내

화면 흐름

신규 등록 (바코드): 01 홈 → 02 방법 선택 → 03 바코드 스캔 → 04 상품+재고 폼 → 01 홈
신규 등록 (수동): 01 홈 → 02 방법 선택 → 05 수동 입력 폼 → 01 홈
기존 상품에 재고 추가: 01 홈 → 02 방법 선택 → 03 바코드 스캔 → 06 재고 추가 → 07 상세
검색 후 수정: 01 홈 → 10 검색 → 07 상세 → 08 재고 수정 → 07 상세
상품 삭제: 07 상세 → 09 삭제 확인 → 01 홈
